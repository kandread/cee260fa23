Expectation

The expected value for a discrete random variable \(X\) taking on possible values {x1, x2,…, xn} is defined by \[E[X]=\sum_i x_i P(X=x_i)\]

Expectation of a continuous variable

The expected value of a continuous random variable \(X\) is defined by \[E[X] = \int_{-\infty}^{\infty} x f(x) dx\]

Example

Calculate the expected value of a random variable \(X\) with PDF \[f(x) = \begin{cases} 1/1.5 & 0 < x < 1.5 \\ 0 & \text{otherwise} \end{cases}\]

What about the expected value of a random function?

  • A random function \(g(X)\) is a function of a random variable \(X\)
  • Each evaluation of the function will have the probability associated with the corresponding value of the random variable
  • Distribution of \(g(X)\) allow calculation of all relevant properties

Example

The time (hrs) to repair an electrical breakdown in a factory is a random variable (\(X\)) whose PDF is \[f_X(x) = \begin{cases} 1 & 0 < x < 1 \\ 0 & \text{otherwise} \end{cases}\] If the cost involved for a breakdown lasting \(x\) is \(x^3\), what is the expected cost of a breakdown?

If \(X\) is a continuous random variable with PDF \(f(x)\) then \[E[g(X)]= \int_{-\infty}^{\infty} g(x) f(x) dx\]

Expected value of sums of random variables

\[ E[X+Y] = \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} (x+y) f(x+y) dx dy \]

\[E[X+Y] = \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} x f(x+y) dx dy + \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} y f(x+y) dx dy\]

E[X+Y] = E[X] + E[Y]

The expected value of the sum of any number of random variables is \[E[X_1 + X_2 + ... + X_n] = E[X_1] + E[X_2] + ... + E[X_n]\]

Covariance

Similar in concept to correlation

The covariance of two random variables \(X\) and \(Y\) is defined as \[\mathrm{Cov}(X, Y) = E[(X - \mu_X)(Y - \mu_Y)]\]


  • \(\mathrm{Cov}(X,Y) = \mathrm{Cov}(Y,X)\)
  • \(\mathrm{Cov}(X,X) = \mathrm{Var}(X)\)

Additive property

\[\mathrm{Cov}(X_1 + X_2, Y) = \mathrm{Cov}(X_1, Y) + \mathrm{Cov}(X_2, Y)\]


\[\mathrm{Cov} \left( \sum_{i=1}^n X_i, Y \right) = \sum_{i=1}^n \mathrm{Cov}(X_i, Y)\]

Variance and covariance

\[\mathrm{Var}\left( \sum_{i=1}^n X_i \right) = \sum_{i=1}^n \mathrm{Var}(X_i) + \sum_{i=1}^n \sum_{j=1, i \neq j}^n \mathrm{Cov}(X_i, X_j)\]

Independent random variables

If \(X\) and \(Y\) are independent random variables, then \[\mathrm{Cov}(X,Y)=0\] \[\mathrm{Var}\left( \sum_{i=1}^n X_i \right) = \sum_{i=1}^n \mathrm{Var}(X_i)\]

Correlation

The correlation between two random variables \(X\) and \(Y\) is defined by \[\mathrm{Corr}(X, Y) = \dfrac{\mathrm{Cov}(X, Y)}{\sqrt{\mathrm{Var}(X) \mathrm{Var}(Y)}}\]
