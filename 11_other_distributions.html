  • Other special random variables that can be derived from a Bernoulli sequence or Poisson process
  • Negative binomial

Negative binomial distribution

A generalization of the geometric distribution where the random variable \(X\) describes the number of Bernoulli trials required to obtain \(r\) successes

  • Probability that \(r-1\) successes in the first \(x-1\) trials is \[\binom{x-1}{r-1}p^{r-1}(1-p)^{x-r}\]
  • Probability that trial \(x\) is a success is \(p\)

In a series of Bernoulli trials (independent trials with constant probability \(p\) of a success), the random variable \(X\) that equals the number of trials until \(r\) successes occur is a negative binomial random variable with parameters \(0 < p < 1\) and $r = 1, 2, 3, …$ \[f(x) = \binom{x-1}{r-1}(1-p)^{x-r}p^r\] where $x=r,r+1,r+2,…$

Mean and variance

If \(X\) is a negative binomial random variable with parameters \(p\) and \(r\), then \[\mu=E[X]=\dfrac{r}{p}\] \[\sigma^2=Var(X)=\dfrac{r(1-p)}{p^2}\]

Hypergeometric distribution

  • 850 manufactured parts contain 50 non-conforming to customer specs
  • Two parts are selected at random without replacement
  • \(A\) and \(B\) are events that first and second parts are non-conforming
  • \(P(A)=50/850\)
  • \(P(B|A)=49/849\)

Let \(X\) be the number of non-conforming parts

\[P(X=0)=P(\text{both parts conform})=\dfrac{800}{850}\dfrac{799}{849}=0.886\]

\[P(X=1)=\dfrac{800}{850}\dfrac{50}{849} + \dfrac{50}{850}\dfrac{849}=0.111\]

\[P(X=2)=\dfrac{50}{850}\dfrac{49}{849}=0.003\]

Hypergeometric distribution

A set of \(N\) objects contains \(K\) successes and \(N-K\) failures. A sample of size \(n\) objects is selected randomly. The random variable \(X\) that equals the number of successes in the sample is a hypergeometric random variable and \[f(x)=\dfrac{\binom{K}{x}\binom{N-K}{n-x}}{\binom{N}{n}}\]

Mean and variance

If \(X\) is a hypergeometric random variable with parameters \(N\), \(K\), and \(n\), then \[\mu=E[X]=np\] \[\sigma^2=Var(X)=np(1-p)\left( \dfrac{N-n}{N-1} \right)\]
