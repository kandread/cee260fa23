Uniform distribution

A random variable \(X\) is uniformly distributed over the interval \([a,b]\) if its probability density function is given by \[f(x) = \begin{cases} \dfrac{1}{b-a} & a \leq x \leq b \\ 0 & \text{otherwise} \end{cases}\]

  • When \(X\) is equally likely to be near any value in \([a,b]\)
  • Examples of applications include quantization error and random number generation in software

Mean and variance

\[E[X] = \int_a^b \dfrac{x}{b-a}dx = \dfrac{b^2-a^2}{2(b-a)}\]

\[= \dfrac{(b-a)(b+a)}{2(b-a)} \Rightarrow E[X] = \dfrac{a+b}{2}\]

\[Var(X) = \dfrac{(b-a)^2}{12}\]

Gamma function

\[\Gamma(r) = \int_{0}^{\infty} x^{r-1}e^{-x}dx \text{, for } r>0\]

\[\Gamma(r)=(r-1)\Gamma(r-1)\] \[\Gamma(1)=1\] \[\Gamma(0.5)=\sqrt{\pi}\]

Gamma distribution

The random variable with probability density function \[f(x)=\dfrac{\lambda^r x^{r-1} e^{-\lambda x}}{\Gamma(r)}\] is a gamma random variable with parameters \(\lambda>0\) and \(r>0\).

  • Consider a sequence of events and the waiting time for each event is exponentially distributed
  • The waiting time for the $n$-th event follows the gamma distribution
  • Application examples include the
    • waiting time until a repair is needed for a hydraulic system
    • size of insurance claims
    • rainfall
    • fading of wireless communication signal

Mean and variance

If \(X\) is a gamma random variable with parameters \(\lambda\) and \(r\) then \[\mu=E(X)=\dfrac{r}{\lambda}\] \[\sigma^2=V(X)=\dfrac{r}{\lambda^2}\]

Incomplete gamma function

\[\gamma(u,r)=\dfrac{\int_{0}^{u}y^{r-1}e^{-y}dy}{\Gamma(r)}\]

\[P(a < X < b) = P(X < b) - P(X < a)\]

\[= \dfrac{1}{\Gamma(r)} \left( \int_{0}^{\lambda b} y^{r-1} e^{-y} dy - \int_{0}^{\lambda a} y^{r-1} e^{-y} dy \right)\]

\[= \gamma(\lambda b, r) - \gamma(\lambda a, r)\]

Beta distribution

  • A flexible distribution that is bounded over a finite range

  • E.g., proportion of solar radiation absorbed by a material

    The random variable \(X\) with probability density function \[f(x) = \dfrac{\Gamma(\alpha + \beta)}{\Gamma(\alpha) \Gamma(\beta)} x^{\alpha - 1} (1 - x)^{\beta - 1}\] for \(0 \leq x \leq 1\) is a beta random variable with parameters \(\alpha>0\) and \(\beta>0\).

Mean and variance

If \(X\) has a beta distribution with parameters \(\alpha\) and \(\beta\) then \[\mu = E(X) = \dfrac{\alpha}{\alpha + \beta}\] \[\sigma^2 = V(X) = \dfrac{\alpha \beta}{(\alpha + \beta)^2 (\alpha + \beta + 1)}\]

Chi-square distribution

If \(Z_1\), \(Z_2\),…, \(Z_n\) are independent standard normal variables, then \(X=\sum_{i=1}^n Z_i\) is said to have a \(\chi^2\) distribution with \(n\) degrees of freedom

  • Has additive property
  • If \(X_1\), \(X_2\) with \(n_1\), \(n_2\) d.o.f. are \(\chi^2\) then \(X_1+X_2\) is \(\chi^2\) with \(n_1+n_2\) d.o.f.
  • Goodness of fit test
  • Confidence intervals for population variance

Not symmetric!

Mean and variance

\(\chi^2\) distribution is identical to the gamma distribution for \(r=n/2\) and \(\lambda=1/2\)

The mean and variance for a random variable \(X\) that follows a \(\chi^2\) distribution are \[E[X]=n\] \[\text{Var}(X)=2n\]

t-distribution

  • The t-distribution generalizes the standard normal distribution
  • Also symmetric around zero and bell-shaped
  • Used in statistical tests for the difference between population means and linear regression

If \(Z\) is a standard normal random variable, and \(\chi_n\) follows a \(\chi^2\) distribution with \(n\) d.o.f., then the random variable defined by \[T_n = \dfrac{Z}{\sqrt{\chi^2}/n}\] is said to have a t-distribution with \(n\) degrees of freedom

Mean and variance

The mean and variance of \(T_n\) are \[E[T_n] = 0\] \[\text{Var}(T_n) = \dfrac{n}{n-2}\]
