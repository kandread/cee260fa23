Normal distribution

A random variable \(X\) with probability density function

\[f(x)=\dfrac{1}{\sqrt{2 \pi \sigma}} e^{-\dfrac{(x-\mu)^2}{2\sigma^2}}\] is a normal random variable with parameters \(\mu\) and \(\sigma\)


What is the distribution of \(Y=a+bX\)?

If \(b \neq 0\) then for a normal random variable \(X\), \(Y=a+bX\) is also a normal random variable with parameters \[E[Y]=a+bE[X]=a+b\mu\] \[\text{Var}(Y)=b^2\text{Var}(X)=b^2\sigma^2\]

Let's verify this!

Problem 13.1

The power \(W\) dissipated in a resistor is proportional to the square of the voltage \(V\) \[W=rV^2\] where \(r\) is a constant. If \(r=3\) and \(V\sim\text{N}(6,1)\), find \(E[W]\) and \(P(W>120)\).

Sum of independent normal variables

For two statistically independent random variables \(X\sim\text{N}(\mu_x,\sigma^2_x)\) and \(Y\sim\text{N}(\mu_y,\sigma^2_y)\), the random variable \(X+Y\) is also normally distributed with parameters \(\mu_{x+y}=\mu_x + \mu_y\) and \(\sigma^2_{x+y}=\sigma^2_x+\sigma^2_y\)

For any linear combination of normal random variables \(X_i\sim\text{N}(\mu_{x_i},\sigma^2_{x_i})\) then \(W=\sum_{i=1}^n a_i X_i\) is also normally distributed with mean \[\mu_w = \sum_{i=1}^n a_i \mu_{x_i}\] and variance \[\sigma^2_w = \sum_{i=1}^n a_i^2 \sigma^2_{x_i}\]

Lognormal distribution

  • Another popular distribution where the variable of interest \(X\) can be transformed to a normally-distributed variable
  • \(\ln{X}\) is a Gaussian random variable
  • Lifetime of a product that degrades over time often modeled with lognormal

If \(W\) is a normally-distributed variable with mean \(\theta\) and variance \(\omega^2\), then \(X\) with \(\ln{X}=W\) is a lognormal random variable with probability density function \[f(x)=\dfrac{1}{x \omega \sqrt{2 \pi}} e^{-\dfrac{(\ln{x}-\theta)^2}{2\omega^2}}\]

The mean and variance of \(X\) are \[E[X]=e^{\theta + \dfrac{\omega^2}{2}}\] \[\text{Var}(X)=e^{2\theta + \omega^2}\left( e^{\omega^2} - 1 \right)\]

Calculating lognormal probabilities

\[P(a < X < b) = \int_a^b \dfrac{1}{x \omega \sqrt{2\pi}} e^{\left[-\dfrac{1}{2} \left(\dfrac{\ln{x}-\theta}{2\omega}\right)^2\right]}\]

Change variables to \(s=\dfrac{\ln{x}-\theta}{\omega}\) and we have

\[=\dfrac{1}{2\pi} \int_{(\ln{a}-\theta)/\omega}^{(\ln{b}-\theta)/\omega} e^{-(1/2)[(s-0)/1]^2} ds = \Phi \left( \dfrac{\ln{b}-\theta}{\omega} \right) - \Phi \left( \dfrac{\ln{a}-\theta}{\omega} \right)\]

Exponential distribution

  • Poisson distribution
  • Number of occurrences anywhere in space or time
  • Example: number of flaws along a length of copper wire
  • What if the random variable \(X\) was the length from a starting point on the wire until a flaw is detected?
  • Distance to the first flaw exceeds \(x\) mm if there are no flaws within \(x\) mm
  • \(N\) is the number flaws in \(x\) mm of wire
  • \(\lambda\) is the mean number of flaws per mm

\[P(X>x)=P(N=0)=\dfrac{e^{-\lambda x} (\lambda x)^0}{0!}=e^{-\lambda x}\]

\[F(x)=P(X \leq x)=1 - e^{-\lambda x}\]

\[f(x)=\dfrac{dF(x)}{dx}=\lambda e^{-\lambda x}\]

The random variable \(X\) that is equal to the distance/time between successive events from a Poisson process with mean number of events \(\lambda > 0\) per unit interval is an exponential random variable with a probability density function \[f(x)=\lambda e^{-\lambda x}\]

Mean and variance

If the random variable \(X\) has an exponential distribution with parameter \(\lambda\) then \[\mu = E(X) = \dfrac{1}{\lambda}\] \[\sigma^2 = V(X) = \dfrac{1}{\lambda^2}\]
