Random variables

  • Sample space and events for simple cases: outcomes
  • Sometimes we need an analytical form to describe these
  • Variable that associates outcomes with real numbers

A random variable is a function that assigns a real number to each outcome in the sample space of a random experiment

Notation

\[X: \text{random variable}\] \[x: \text{value of random variable}\]

Any event can be expressed as \(X=a\), or \(X \leq a\), or \(a < X \leq b\)

Mapping of events

Example

  • Two electronic components that could either be defective (d) or acceptable (a)
  • P[(d,d)]=0.09, P[(d,a)]=0.21, P[(a,d)]=0.21, P[(a,a)]=0.49
  • \(X\): the number of acceptable components
  • \(P(X=0)=0.09\), \(P(X=1)=0.42\), \(P(X=2)=0.49\)
  • \(A\): event of at least one acceptable component
  • \(I\): indicator variable of \(A\)
  • \(P(I=0)=0.09\) and \(P(I=1)=0.91\)

Discrete and Continuous random variables

  • Discrete and continuous sample space
  • Discrete random variable
    • random variable with a finite (or countably infinite) range
  • Continuous random variable
    • random variable with an interval (either finite or infinite) of real numbers for its range

Examples of random variables

  • Electrical current
  • Length
  • Pressure
  • Number of surface scratches
  • Proportion of defective parts among 1000 tested

Probability distributions

Mathematically describing the probability of values or ranges of a random variable

The cumulative distribution function (or distribution function) \(F\) or a random variable \(X\) can be defined as \[F(x) \equiv P(X \leq x)\]

Probability rules apply here too!

How can we calculate the probability of \(a < X \leq b\) in terms of \(F\)?

\[\{ X \leq b \} = \{ X \leq a \} \cup \{ a < X \leq b \}\]

\[P(X \leq b) = P(X \leq a) + P(a < X \leq b)\]

\[P(a < X \leq b) = F(b) - F(a)\]

For a random variable \(X\), the distribution function is \[F(x)\begin{cases} 0 & x \leq 0 \\ 1-e^{-x^2} & x > 0 \end{cases}\]

What is the probability that \(X>1\)?

Probability Mass Function

For a discrete random variable \(X\), the probability mass function (PMF) describes the probability that \(X=a\) \[p(a)=P(X=a)\]

If \(X\) takes only values in {x1, x2,…} then \[\begin{cases}p(x_i)>0 & i=1,2,... \\ p(x)=0 & \text{all other values}\end{cases}\]

\[\sum_{i=1}^{\infty} p(x_i) = 1\]

The cumulative distribution function can be expressed as \[F(a)=\sum_{\forall x \leq a} p(x)\]

Probability Density Function

For a continuous random variable \(X\), the probability density function (PDF) \(f(x)\) is a non-negative function defined for \(x \in (-\infty, \infty)\) \[P(X)=\int_{-\infty}^{\infty} f(x)dx\]

\[P(a \leq X \leq b) = \int_{a}^{b} f(x) dx\]

The cumulative distribution function (CDF) will be \[F(a)=P(X \leq a)=\int_{\infty}^{a} f(x) dx\]

If we differentiate \[\dfrac{d}{da}F(a) = f(a)\]

What is the probability that \(P(x=a)\)?

Joint probability distributions

The probability distribution that defines the simultaneous behavior of two or more random variables

Joint cumulative probability distribution function

\[F(x,y)=P(X \leq x, Y \leq y)\]

The distribution function of \(X\) would be \[F_X(x) = P(X \leq x) = P(X \leq x, Y \leq \infty) = F(x,\infty)\]

Joint probability mass function

The joint PMF of the discrete random variables \(X\) and \(Y\), denoted as \(f_{XY}(x, y)\), satisfies \[f_{X,Y}(x, y) \geq 0\] \[\sum_X f_{XY}(x,y) = 1\] \[f_{XY}(x,y) = P(X=x, Y=y)\]

How do we find the marginal probability?

\[P(X=x_i)?\]

\[\{X=x_i\}= \bigcup_j \{X=x_i, Y=y_j\}\]

\[P(X=x_i) = \sum_j P(X=x_i, Y=y_j)\]

\[P(X=x_i) = \sum_j p(x_i, y_j)\]

Joint probability density function

A joint PDF for the continuous random variables \(X\) and \(Y\), denoted as \(f_{XY}(x,y)\), satisfies \[f_{XY}(x,y) \geq 0\] \[\int_{-\infty}^{\infty} \int_{-\infty}^{\infty} f_{XY}(x,y) dx dy = 1\] \[P[(X,Y) \in \mathbb{R}] = \int \int_{\mathbb{R}} f_{XY}(x,y) dx dy\]

Independent random variables

Random variables \(X\) and \(Y\) are said to be independent if for any two sets of real numbers \(A\) and \(B\) \[P(X \in A, Y \in B)=P(X \in A)P(X \in B)\]


For discrete variables \(X\) and \(Y\), independence is equivalent to \[p(x,y)=p_X(x)p_Y(y)\,\,\forall x,y\]

For any two sets \(A\) and \(B\) \[P(X \in A, Y \in B)= \sum_{y \in B} \sum_{x \in A} p(x,y)\]

\[= \sum_{y \in B} \sum_{x \in A} p_x(x)p_Y(y) = \sum_{y \in B} p_Y(y) \sum_{x \in A} p_X(x)\]

\[= P(Y \in B) P(X \in A)\]

For continuous variables \[f(x,y)=f_X(x) f_Y(y)\]

\[P(X \leq a, Y \leq b) = P(X \leq a)P(Y \leq b)\]

\[F(a, b) = F_X(a) F_Y(b)\]

Conditional distributions

Recall that \(P(A|B)=\dfrac{P(AB)}{P(B)}\)

For discrete variables we have \[p_{X|Y}(x|y) = P(X=x|Y=y) = \dfrac{P(X=x,Y=y)}{P(Y=y)} = \dfrac{p(x,y)}{p(y)}\]


For continuous variables we have \[f_{X|Y}(x|y)=\dfrac{f(x,y)}{f_Y(y)}\] \[P(X \in A| Y = y) = \int_A f_{X|Y}(x|y) dx\]
