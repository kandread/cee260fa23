Testing hypotheses on regression

  • As we know the statistical properties of the regression parameters, we can test hypotheses
  • Or calculate probabilities, confidence intervals etc.
  • Important hypothesis to test: \(\beta=0\)?
  • If \(\beta=0\) then there is no regression on the input variable

Distribution of \(\beta\) test statistic

Remember that \(B \sim \text{N}\left( \beta, \sigma^2/S_{xx} \right)\)

\[\sqrt{\dfrac{(n-2)S_{xx}}{SS_E}} (B - \beta) \sim t_{n-2}\]

Hypothesis test of \(H_0:\;\beta=0\)

reject \(H_0\) if \(\sqrt{\dfrac{(n-2)S_{xx}}{SS_E}}|B| > t_{\gamma/2,n-2}\)

accept \(H_0\) otherwise/

Hypothesis test of \(H_0:\;\alpha=a\)

reject \(H_0\) if \(\sqrt{\dfrac{n(n-2)S_{xx}}{SS_E}\sum_i x_i^2}(A-a) > t_{\gamma/2,n-2}\)

accept \(H_0\) otherwise/

Confidence intervals on regression

Confidence interval for \(\beta\)

A \(100(1-\gamma)\) percent confidence interval estimator of \(\beta\) is

\[B \pm \sqrt{\dfrac{SS_E}{(n-2)S_{xx}}} t_{\gamma/2, n-2}\]

Confidence interval for \(\alpha\)

The \(100(1-\gamma)\) percent confidence interval for \(\alpha\) is

\[A \pm t_{\gamma/2,n-2} \sqrt{\dfrac{SS_E \sum_i x_i^2}{n(n-2)S_{xx}}}\]

Confidence interval for \(\alpha + \beta x_0\)

With \(100(1-\gamma)\) percent confidence \(\alpha+\beta x_0\) will lie within

\[A + B x_0 \pm \sqrt{ \dfrac{1}{n} + \dfrac{(x_0 - \overline{x})^2}{S_{xx}}} \sqrt{\dfrac{SS_E}{n-2}} t_{\gamma/2,n-2}\]

Adequacy of the regression model

  • Fitting a regression model requires several assumptions
  • Uncorrelated errors with zero mean and constant variance
  • When testing hypotheses, errors are normally distributed
  • Relationship between variables is linear
  • Need methods to evaluate these assumptions

Residual analysis

\[e_i=y_i - \hat{y}_i\]

  • Histogram of residuals
  • Normal probability plot

Could also standardize residuals with \[d_i=\dfrac{e_i}{\sqrt{\hat{\sigma}^2}}\] 95% of residuals should fall within (-2, +2)

Coefficient of determination

The coefficient of determination is \[R^2 = \dfrac{SS_{yy}-SS_R}{SS_{yy}} = 1 - \dfrac{SS_R}{SS_{yy}}\]

  • \(0 \leq R^2 \leq 1\)
  • Amount of variance explained by the regression model

Some caveats

  • \(R^2\) can be artificially increased by adding terms in the regression
  • The larger the dispersion of \(x\), the larger the value of \(R^2\)
  • A large value of \(R^2\) does not imply a steep slope
  • A large value of \(R^2\) does not mean that the model extrapolates accurately

Correlation

Remember that we defined the correlation coefficient as \[r = \dfrac{\sum_{i=1}^n (x_i - \overline{x})(y_i - \overline{y})}{\sqrt{\sum_{i=1}^n (x_i - \overline{x})^2 \sum_{i=1}^n (y_i - \overline{y})^2}}\]

\(r\) provides a measure of the degree of linear association between the \(x\) and \(y\) variables

Using the linear regression notation

\[r = \dfrac{S_{xy}}{\sqrt{S_{xx}S_{yy}}}\]

\[r^2 = \dfrac{S^2_{xy}}{S_{xx}S_{yy}} = \dfrac{S_{xx}S_{yy} - SS_R S_{xx}}{S_{xx}S_{yy}} = 1 - \dfrac{SS_R}{S_{yy}}\]

\[|r| = \sqrt{R^2}\]
